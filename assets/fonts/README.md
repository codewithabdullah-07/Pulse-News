# assets/fonts/

Pulse News uses Google Fonts loaded at runtime (via the google_fonts package),
so no font files need to be bundled here for development.

For offline/production use, you can download and bundle the fonts manually:
- DM Sans: https://fonts.google.com/specimen/DM+Sans
- Playfair Display: https://fonts.google.com/specimen/Playfair+Display

Then reference them in pubspec.yaml under the fonts: section (already pre-configured).

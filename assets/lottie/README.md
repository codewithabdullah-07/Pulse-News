# assets/lottie/

Place Lottie animation JSON files here.
You can download free animations from https://lottiefiles.com

Suggested additions for Pulse News:
- loading.json  → used on fetch/refresh screens
- empty.json    → used for empty bookmarks / no results
- error.json    → used for error state

Reference in code: Lottie.asset('assets/lottie/loading.json')
